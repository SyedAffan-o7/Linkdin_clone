# Linkdin_clone

A LinkedIn Feed clone built with React, Vite, and Tailwind CSS.

## Features

- **Navbar** - LinkedIn-style navigation with Home, Network, Jobs, Messaging, Notifications
- **Left Sidebar** - Profile card, stats, recent items, groups, events, hashtags
- **Main Feed** - Create posts, view posts with like/comment/share actions
- **Right Sidebar** - LinkedIn News, Premium promotion, suggested courses, add to feed
- **Sticky Layout** - Sidebars stay fixed while only the feed scrolls
- **Hidden Scrollbars** - Clean UI with functional scrolling but no visible scrollbars

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 to view the app.
